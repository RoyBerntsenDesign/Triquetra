#Triquetra README

Print Settings:
Material: CPE, PETG or some hight tg and low shrink material 

Linewidth: 0.5
Layerheight: 0.1, 0.2 or 0.25

Shell must be 2mm:
Top, bottom layers should be 2mm each (example 0.2 x 10)
Walls should be 2mm (example 0.5 x 4)
